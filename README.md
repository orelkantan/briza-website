# 🔄 אוטומציית ביקורות גוגל — בריזה (גרסה מעודכנת)

מערכת אוטומטית למשיכה, צבירה והצגת ביקורות מ-Google Maps ישירות לאתר "בריזה".

---

## 🚀 איך זה עובד?

1. **תזמון:** כל יום ב-**00:00 (שעון ישראל)**, ה-GitHub Action מתעורר.
2. **משיכה:** הסקריפט `update_reviews.py` פונה ל-Google Places API ומושך את 5 הביקורות האחרונות.
3. **צבירה:** ביקורות חדשות מתווספות לקובץ `.reviews_cache.json` (המאגר הדינמי שלנו).
4. **הזרקה:** הסקריפט מחפש ב-`index.html` את האלמנט עם ה-ID `reviews-track` ומזריק לתוכו את כל הביקורות המעודכנות.
5. **פרסום:** GitHub Pages מעדכן את האתר החי באופן אוטומטי.
